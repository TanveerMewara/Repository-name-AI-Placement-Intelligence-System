# AI Placement Intelligence System

## Overview

AI Placement Intelligence System is an end-to-end machine learning and NLP-based platform that helps students evaluate their placement readiness.

The system can:

* Extract text from resumes
* Identify technical skills using NLP
* Compare resumes with job descriptions
* Calculate skill match scores
* Predict placement outcomes using Machine Learning
* Store assessment data in SQL
* Visualize analytics through an interactive dashboard

---

## Features

### Resume Parsing

Upload a PDF resume and extract text automatically.

### Skill Extraction

Identify relevant technical skills from the resume.

### Job Description Matching

Compare candidate skills with job requirements and calculate a match percentage.

### Placement Prediction

Predict placement outcomes using a Random Forest Classifier.

### SQL Database Storage

Store candidate assessments using SQLite.

### Analytics Dashboard

Visualize placement trends and dataset insights using Matplotlib and Seaborn.
